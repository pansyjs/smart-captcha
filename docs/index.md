## 🏗 安装

```bash
# npm install
npm install @pansy/smart-captcha --save

# yarn install
yarn add @pansy/smart-captcha

# pnpm install
pnpm i @pansy/smart-captcha
```

## 🔨 使用

```ts

export default () => {
  <SmartCaptcha />
}
```
