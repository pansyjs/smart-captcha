## API

|属性|描述|类型|默认值|
|---|---|----|-----|
|className|节点类名|`string`|-|
|style|节点样式|`CSSProperties`|-|
|elementId|渲染的目标元素ID|`string`|-|
|width|组件宽度|`number`|`300`|
|height|组件高度|`number`|`42`|
|onSuccess|验证成功回调|`(data) => void`|-|
|onChange|验证成功/失败回调|`(data) => void`|-|
|onFailed|验证失败回调|`() => void`|-|
